# Target Users

## Primary Target Users

SkillForge is made for students and people who want to become professionals. They need help to achieve their career goals.

---

# User Persona 1

## Undergraduate Student


## Profile

- User is a second-year Computer Science student

- Knows the basics of Python

- Wants to become a Data Scientist

## Challenges

- User does not know what to learn

- Is confused by all the online roadmaps

- Is not sure if he is ready for a job in the industry

## Goals

- User wants to become a Data Scientist

- Wants to build a strong portfolio

- Wants to get an internship

## How SkillForge Helps

- SkillForge identifies the skills User is missing

- It generates a roadmap that is just for user

- SkillForge recommends things to learn and projects to work on

# User Needs Summary

Users of SkillForge expect it to:

- Tell them what skills they need for the career they want

- Look at the skills they already have

- Show them what skills they are missing

- Give them a plan to learn what they need

- Suggest good things to learn

- Track how they are doing over time

- Help them get ready, for a career.
